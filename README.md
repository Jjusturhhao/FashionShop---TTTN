# FashionShop---TTTN
Graduation Internship Project
DATA ANALYSIS PROJECT
Fashion Shop Sales Analysis

## 1. Project Overview

This project analyzes a fashion retail sales dataset to understand sales performance, customer behavior, and product trends. The goal is to transform raw sales data into meaningful insights using data cleaning techniques, SQL queries, and data visualization.

### The project includes:
- Data cleaning using SQL
- Data exploration and analysis
- Data visualization using Power BI
- Business insights derived from the dashboard

## 2. Dataset Description

The dataset used in this project contains sales information from a fashion shop. The data includes multiple tables related to customers, products, orders, and transactions.

### Main information included in the dataset:
- Customer information
- Product categories
- Sales transactions
- Order details
- Revenue data

These datasets are used to analyze business performance and identify important patterns in sales activities.

## 3. Data Cleaning

Before performing analysis, the raw dataset was cleaned to ensure data accuracy and consistency.

### The data cleaning process includes:
- Removing duplicate records
- Handling missing values
- Standardizing data formats
- Creating calculated columns for analysis
- Validating relationships between tables

SQL was used to transform and clean the data to make it suitable for analysis.

## 4. Data Analysis

After cleaning the data, exploratory data analysis was conducted to identify key patterns in the dataset.

### Some of the analysis tasks include:
- Analyzing total sales revenue
- Identifying best-selling products
- Analyzing customer purchasing behavior
- Evaluating sales performance across product categories
- Identifying trends in sales over time
- SQL queries were used to extract insights from the dataset.

## 5. Dashboard Visualization

Power BI was used to build an interactive dashboard that visualizes key business metrics.

### The dashboard includes several sections:

- Sales Overview: Provides a summary of total revenue, total orders, and sales performance over time.

- Product Analysis: Shows the best-selling products and product categories contributing the most revenue.

- Customer Analysis: Analyzes customer purchasing behavior and identifies important customer segments.

The dashboard helps stakeholders quickly understand business performance and key trends.

## 6. Key Insights

From the analysis, several important insights were identified:

- Certain product categories generate the majority of sales revenue.

- A small group of customers contributes a significant portion of total sales.

- Sales performance varies across different time periods.

- Some products consistently perform better than others.

These insights can help businesses improve marketing strategies, optimize product offerings, and increase overall sales performance.

## 7. Tools Used

The following tools were used in this project:

- SQL – Data cleaning and querying

- Excel – Data storage and preprocessing

- Power BI – Dashboard visualization

## 8. Conclusion

This project demonstrates the process of transforming raw business data into meaningful insights through data cleaning, analysis, and visualization. The results provide valuable information that can support business decision-making in retail sales management.
